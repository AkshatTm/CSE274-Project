## 2024-03-24 - Typography and Accessibility Improvements\n**Learning:** The application's UI benefits greatly from a serif/sans-serif typography combination (Playfair Display + Inter) to break away from a generic dashboard aesthetic while maintaining readability. React's strict purity rules require careful handling of functions like `Math.random()` within functional components, effectively addressed by memoizing data generation.\n**Action:** When updating typography, ensure font variables are consistently mapped across the CSS and inline styles. Always memoize random data generation in React components to avoid purity linting errors.
